Activity 1
----------

The program has been refactored by introducing JSP 'DisplayResultView.jsp' as the View thus changing the sample web application from Model 1 pattern to MVC Model 2 pattern.
 
Start the Tomcat server. Now, go to the URL http://localhost:8080/lab3part1/booktown and press Enter. The user can create a new author by iputting Last name and first name and then clicking button 'Create author'. Similary, user can delete the existing author's that are displayed in the List by clicking the Delete action.
